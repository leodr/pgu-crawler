> This Node.js program crawls the news, calendar and canteen pages from http://www.pgu.de/ and stores them in Cloud Firestore collections.

### 🏠 [Homepage](https://github.com/leodr/pgu-crawler)

## Prerequisites

- node >=12.14.0

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```
